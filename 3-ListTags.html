List Tags

There are 3 types of list tags;

1-Ordered List "ol" is used to create numerated lists. We use this with list item "li" tag. "li" is used to add item to the mentioned list.
  1. Istanbul
  2. Ankara
  3. Antalya
Note 1: How do I change numbers to letters. We have a command called attributer. Attributers change the features of the tag. If "ol" is used w/out anything, as a default, it comes with numbers. If I change it to "ol type="a"'' Type attributer "type" gives different features to the tag I created. the types I can change to; types = 1, A, a, I, i See below;
  1. Istanbul
  2. Ankara
  3. Antalya
Note 2: Starting Point; We can change the starting point of the list with "start". We have use number for the start attribute. The number is equivalence of the character in natural order. For example type="a" start="3" will be c
