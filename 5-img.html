IMAGE

Note1

We write "img" and click. That brings us "img src="" alt="" -> src; is the source of our image and we write ./ to choose the link for that file. Where it says alt="", we write an explanation text for that image. This is necessary if the image isnt openned or if a blind person might use the read option to explore the image.

Note2

If we want to change the dimensions we add width and height attributes to our tag -> "img src="./assets/img/Screenshot 2023-04-04 at 08.51.10.png" alt="Gallery Wall Pictures" width="1", height="auto" when we say "auto" to one of the dimensions, it changes the dimension in a ratio with the other dimension

Note3

SEO -> Search engine optimization alt is used in the img tag is important for SEO purposes. One of the places search engines look for creating indexes
