Core CSS

CSS is the language we use to style a Web page.

1.What is CSS?

CSS manages the colors, fonts, and layouts of your website elements, as well as allowing you to add effects or animations to your pages.
  • CSS stands for Cascading Style Sheets
  • CSS describes how HTML elements are to be displayed on screen, paper, or in other media
  • An HTML element is defined by a start tag, some content, and an end tag.
  • CSS saves a lot of work. It can control the layout of multiple web pages all at once
  • External stylesheets are stored in CSS files

2. Styling HTML

There are 3 ways of styling HTML
  • Inline styling
  • Internal styling
  • External styling

2.1- Inline styling

We open our HTML file and use the styling inside individual HTML tags. For example; " style="color : red">Hello World" -> in in the tag where it says color : red is CSS and changes text colors. 1- color -> is used to change the text color -> " style="color : red">Hello World" 2- background-color -> is used to change the background color -> " style="background-color : red">Hello World" 3- font-size -> is used the changed the font size. For example, I can make my h1 so big or so small..For SEO h1 is important. I may want a very small headline, n that case it'd be useful. "a href="page1.html" style="font-size:20px""Hello World"/a" 4- text-align -> is used to put the text left, right, center, justify "div style="text-align:center""Hello World"/div"

2.2- Internal Styling

When we use "style" tag, this effects whole page. It is usually used in "head" tag "style p{ color: red; text-align:center; font-size: 22px; } /style"

2.3- External Styling

If we have some styles that are used for more than one html page, we save them in a folder. We create a folder called style.css and connect to the other html files where styles are used. "link rel="stylesheet" href="./CoreCSS.html""" link is a tag. rel is a relation. we say connect my stylepage to the html page we want to relate to...we can relate this stylesheet to as many page as we want.

  2. Select and change the elements with the same tag

You go to your html file, go to the paragraph and write the following syntax -> "p class="xyz""............"p" We created a class here.   4. id attribute => Select only one element
    5. The HTML id attribute is used to specify a unique id for an HTML element. You cannot have more than one element with the same id in an HTML document.

Select Summary

We have 4 CSS selections; * for evertyhing, tag for the tags on the page, . for class, # for id. A Class name can be used by multiple HTML elements, while an ID name must only be used by one HTML element within the page

Height and Width
The CSS height and width properties are used to set the height and width of an element. The CSS max-width property is used to set the maximum width of an element. The height and width properties do not include padding, borders, or margins. It sets the height/width of the area inside the padding, border, and margin of the element.

Measurement Units

1- Absolute length units; cm, mm, in, px
2- Relative length units
%, em(Font size of the parent), rem(Font size of the root element), vh, vw
NotePPI stands for pixels per inch. This is what digital resolution is measured in. Most monitors will display images in the neighborhood of 67-100ppi. The standard resolution for web images is 72ppi. This will display your image so that it looks crisp and sharp on screen.
em vs rem
Within CSS, em and rem are both scalable units that also specify values of properties. em and rem meet web accessibility standards, and, unlike px, scale better. Consequently, they are more suited for responsive design.
