Anchor Tag

"a" stands for anchor tag. The tag defines a hyperlink, which is used to link from one page to another. The most important attribute of the element is the href attribute , which indicates the link's destination. Note 1: href -> hypertext reference It is used;

  • link from one page to another
  • link to different a part on the same page
  • link different part in a different page
  • link to a different wesite,email addresss, phone numbers, whatsapp application

1- Link to another page Click here to go to Index Page
Note 1: When we go to another page or another folder on our computer, after we write a href=" we start with ./ and this code gives us a list, we choose where we want to go from that list.
2-Link to a different part on the same page Click here to go to the end of the page
3-A specific area on another page Go to the 2nd chapter on Example page
4-Link to a website Click here to go to Google.com
Note 2:When we click on the link if want to open a new window, add target="_blank" attribute to the tag -> Click here to open the Google on a new page
